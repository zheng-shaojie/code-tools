🎯 对象类型路径建议功能展示

🔢 数组类型 JSON 数据

[ { "id": 1, "name": "张三", "skills": ["JavaScript", "Vue", "React"], "address": { "city": "北京", "district": "朝阳区" } }, { "id": 2, "name": "李四", "skills": ["Python", "Django"], "address": { "city": "上海", "district": "浦东新区" } } ]
数组元素路径建议:
整个数组
包含 2 个元素
name
每项的 name 字段 - 字符串类型
skills
每项的 skills 字段 - 嵌套数组
address.city
每项的 address.city 字段 - 字符串类型

📦 对象类型 JSON 数据

{ "user": { "id": 12345, "profile": { "name": "王五", "email": "wangwu@example.com", "avatar": "https://example.com/avatar.jpg" }, "settings": { "theme": "dark", "notifications": true, "privacy": { "showEmail": false, "showPhone": true } } }, "permissions": ["read", "write", "admin"], "lastLogin": "2023-12-01T10:30:00Z", "isActive": true }
对象路径建议:
$.user.id
数字类型: 12345
$.user.profile.name
字符串类型: 王五
$.user.profile.email
字符串类型: wangwu@example.com
$.user.settings.theme
字符串类型: dark
$.permissions
数组字段 (包含 3 个元素)
$.isActive
布尔值类型: true

💡 使用指南

如何使用对象类型路径建议功能：

  1. 上传Excel文件：包含对象类型JSON数据的Excel文件
  2. 选择源列：选择包含JSON对象数据的列
  3. 查看类型：系统自动识别为"📦 对象类型"
  4. 点击建议：在JSON路径输入框旁点击"💡 建议"按钮
  5. 选择路径：从下拉列表中选择需要的对象路径
  6. 预览结果：路径自动填入，可以立即预览提取结果

🚀 现在对象类型和数组类型都有智能路径建议功能了！
